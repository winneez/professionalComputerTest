# professionalComputerTest
 Professional Computer Test (Angular 12 with css and primeng)

**1. ทำการติดตั้ง :** <br>

npm install <br>
npm install primeng --save <br>
npm install font-awesome --save <br>
npm install primeicons --save <br>
npm install primeflex <br>

**2. Build Project :** <br>
ng build <br>

**3. Run Project :** <br>
ng serve <br>
or <br>
npm start
